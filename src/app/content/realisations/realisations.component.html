Realisaties

Voor het project 4.0 kregen we van de stad Herentals de opdracht om een proof-of-concept om Herentals te transformeren tot een smartcity. Met deze applicatie kon de stad dan snel en makkelijk de vuilnisbakken doorheen de stad monitoren via ingebouwde sensoren en een microcontroller. Ook konden enquêtes bekeken worden en via een kiosk in de stad worden ingevuld.

De webapplicatie is gemaakt met React, de backend is gemaakt met ASP.NET core die gehost is op Azure en als monitoring voor de vuilbakken werd gebruik gemaakt van een ESP microcontroller. Ik specifiek heb vooral gewerkt aan de backend in ASP.NET core en ook heb ik soms geholpen met de front-end. Hiernaast was ik ook de project leider van onze groep in de realisatiefase.

Als eerste opdracht met een "echte" opdrachtgever, heb ik beter geleerd hoe het ontwikkelen van een oplossing in zijn werk gaat wanneer je in het werkveld stapt. Ook als project leider heb ik meer geleerd over communicatie met een klant/opdrachtgever, hoe je best een team aanstuurt en wat er allemaal bij komt kijken om efficiënt een team te leiden.

Als opdracht voor het opleidingsonderdeel Mobile development maakten een medestudent en ik een app waarmee u in augmented reality notities kunt plakken op een object en daarna ook de notities kunt bekijken, bewerken of verwijderen in het overzicht.

Deze applicatie is gemaakt in flutter waarbij ik vooral bezig ben geweest met het niet AR gedeelte omdat je geen IOS apps kunt maken op een Windows computer en het AR gedeelte dus niet kon testen.

Dit was de eerste grote applicatie die ik gemaakt heb in Flutter en heb dus ook veel bijgeleerd over het framework. Het enige spijtige is dat je niet op IOS kunt debuggen met een Windows computer.

De opdracht die we kregen tijdens de projectweek in het tweede jaar en het opo Project PHP was het ontwikkelen van een site voor het organiseren van een festival. Met deze applicatie kon je als gast de line-up en het nieuws bekijken maar ook na het aanmaken van een account tickets kopen. Natuurlijk waren er ook beheersfuncties voorzien waarmee de administratoren alle items op de site konden bewerken of verwijderen.

In dit groepswerk ben ik aan de slag gegaan als projectleider en heb ik ook verschillende pagina’s zoals 1 van de beheer pagina’s ontwikkeld.

Dit project heeft mij een eerste kennismaking gegeven met leiding geven en een volledig project ontwikklen van analyse tot aflevering aan de klant. Ik denk dan ook dat ik hiermee een grote stap heb gezet om een betere projectmatige werker te worden.

Sommige functionaliteiten kunnen niet getoond worden op deze pagina omwille van de CORS (Cross-origin Resource Sharing) en CSRF (Cross-site Request Forgery) beveiliging in de webapplicatie. Het volledige project vind u hier: projectphp31.sinners.be

Voor deze opdracht maakte ik gebruik van Angular als front-end en ASP.NET core als backend om een nieuws site te ontwikkelen. Functionaliteiten zoals artikels lezen en liken of er op reageren is niet het enige ook kunnen artikels worden beheerd door redacteurs die nieuwe artikels kunnen schrijven en die dan worden goedgekeurd door de hoofdredacteur.

Dit was mijn eerste grote opdracht in een ander front-end framework dan JQuery. Hierdoor heb ik kunnen kennismaken met een moderner framework en hoe nieuwe webapplicaties kunnen ontwikkeld worden.
